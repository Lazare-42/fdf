ORTOGRAPHIC PROJECTION:

To project the 3D point ax, ay, az onto the 2d point bx, by using an orthographic
projection parallel to the y axis (where positive y represents forward direction 
- profile view), the following equations can be used :
bx = sx ax + cx;
by = sz az + cz;
where the vector s is an arbitrary scale factor, and c is an arbitrary offset.
